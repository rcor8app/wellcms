# WellCMS Core

WellCMS is a fast, highly customizable, and lightweight admin panel for Laravel applications, built with Livewire, Tailwind CSS, and Alpine.js.

This package has been refactored to use native custom class styling (`re-` prefixes) and features a free MIT license.

## Requirements

- PHP 8.2 or higher
- Laravel 11.0 or higher
- Livewire 3.0 or higher

## Installation

You can install the package via Composer by adding the VCS repository to your `composer.json` and running:

```bash
composer require wellcms/wellcms
```
